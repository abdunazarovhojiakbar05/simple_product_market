RESTful API Development: Mahsulotlarni boshqarish, xaridlar tarixi va foydalanuvchi tizimi uchun yuqori unumdorlikka ega REST API qatlamini ishlab chiqildi.

Security & Authentication: Tizim xavfsizligini ta'minlash uchun Spring Security va JWT (JSON Web Token) asosidagi autentifikatsiya va avtorizatsiya tizimi noldan qurildi.

Database Management: Ma'lumotlarni saqlash va boshqarish uchun PostgreSQL bazasi loyihalashtirildi hamda ma'lumotlar yaxlitligi (data integrity) ta'minlandi.

Admin Panel Logic: Adminlar uchun yangi mahsulotlar qo'shish, mavjudlarini tahrirlash va barcha xaridlarni kuzatish imkonini beruvchi biznes mantiq (business logic) ishlab chiqildi.

Architectural Patterns: Kodning o'qilishi va kengayuvchanligini oshirish uchun DTO (Data Transfer Object) va Mapper patternlaridan foydalanildi.

Remote Testing & Collaboration: API-larni masofaviy qurilmalarda test qilish uchun Ngrok va LAN tarmoqlari sozlandi va loyiha hamkorlar bilan birgalikda muvaffaqiyatli sinovdan o'tkazildi.

Technologies: Java 21, Spring Boot, Spring Security, JWT, PostgreSQL, Maven, Git.
